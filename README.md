# ML Project: Анализ рынка недвижимости

## О проекте
Глобальная цель проекта — разработать сервис для автоматической проверки объявлений о продаже/аренде квартир и оценки справедливости их стоимости.

## Источник данных
Данные собираются с площадки **mirkvartir.ru** (Мир Квартир). 
В качестве инструмента для сбора используется самописный парсер на базе `requests` и `BeautifulSoup4`.
Собираемые артефакты:
- **Структурированные данные:** цена, площадь, этаж, координаты (широта и долгота).
- **Текстовое описание:** свободный текст объявления от автора.
- **Фотографии:** ссылки на изображения интерьеров и планировок.

## Инфраструктура и настройка окружения
Проект использует современные инструменты управления зависимостями и версионирования:
- Пакетный менеджер: **uv**
- Контроль версий кода: **Git**
- Контроль версий данных: **DVC** (с локальным remote-хранилищем)
